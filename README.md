## 🛠 Quá trình Thực hiện

Dự án này được xây dựng dựa trên nền tảng **React.js** với mục tiêu tạo ra một giao diện chuyên nghiệp, hiện đại và phản hồi tốt trên mọi thiết bị (Responsive Design).

### Các bước triển khai:
1.  **Lên ý tưởng & Thiết kế:** Xác định các phần quan trọng (Giới thiệu, Dự án, Chứng chỉ, Liên hệ).
2.  **Cấu trúc dữ liệu:** Sử dụng file `src/portfolio.js` để quản lý tập trung toàn bộ thông tin cá nhân, giúp việc cập nhật dễ dàng hơn.
3.  **Phát triển UI/UX:** Sử dụng các thành phần (components) React, kết hợp với các hiệu ứng hoạt ảnh (Lottie animations) để tăng tính sinh động.
4.  **Tích hợp SEO:** Tối ưu hóa các thẻ meta, tiêu đề để website có thứ hạng tốt trên công cụ tìm kiếm.
5.  **Deployment:** Tự động hóa quy trình triển khai thông qua **GitHub Actions** lên **GitHub Pages**.
